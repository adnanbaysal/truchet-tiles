# truchet-tiles
A new method to produce interesting patterns with Truchet tiles (https://en.wikipedia.org/wiki/Truchet_tiles).

Base tiles are are similar to the following tiles (we use lines instead of quarter arcs):
![Base 0](https://upload.wikimedia.org/wikipedia/commons/1/12/Truchet_tile.svg)
![Base 1](https://upload.wikimedia.org/wikipedia/commons/e/e9/Truchet_tile_inverse.svg)

Base 1 is 90 degree rotated version  of Base 1, so physically only one type of tile is sufficient to produce the patterns in real world.
